This is a private organization for **Exogen**, a project focused on the integration of AI and blockchain technologies.

---

### Our Repositories

| Repository | Description |
|------------|-------------|
| **frontend** | Web application built with Next.js - Dashboard & user interface |
| **mobile** | React Native mobile application for iOS & Android |
| **API_Cli** | Public-facing API client services |
| **API_Private** | Private API backend services |
| **BotExecStrat** | Trading bot strategy execution engine |
| **BotExecStratCPP** | High-performance C++ trading bot execution |
| **Backtester** | Strategy backtesting and simulation engine |
| **DesignPatern** | Design patterns and architectural documentation |
| **Deploy** | Deployment scripts and configurations |
| **k3s-deploy** | Kubernetes (K3s) deployment manifests |
